# SIM Auto Workshop Symfony

Application Symfony simple en arabe RTL pour un tableau de bord atelier SIM Auto.

## Demarrage avec Docker

```powershell
cd C:\projects\simauto-workshop
docker compose up --build
```

Puis ouvrir :

```text
http://localhost:8090
```

Le conteneur PHP lance `composer install` automatiquement au demarrage.
